# Quickstart

Build a chat app that talks to this page. The widget in the sidebar
is a live XMTP inbox — you'll build a Vite app that sends messages
to it using the same identity.

<QuickstartProvider>

## 1. Set up your project

Create a new Vite project and install XMTP. Select **no** when Vite
asks to install and start:

<QuickstartStep step="install" />

The XMTP SDK uses WASM, which requires a Vite config change.
Add `vite.config.js` to your project root:

```javascript
import { defineConfig } from "vite";

export default defineConfig({
  optimizeDeps: {
    exclude: ["@xmtp/browser-sdk", "@xmtp/wasm-bindings"],
  },
});
```
