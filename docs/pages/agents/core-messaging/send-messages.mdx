# Send messages

Once you have the group chat or DM conversation, you can send messages in the conversation.

```tsx [Node]
// Using the agent-sdk, sending messages is simplified through the context
agent.on("text", async (ctx) => {
  // Send a reply to the same conversation
  await ctx.conversation.send("Hello world");
  
  // Or send a reply with helper method
  await ctx.sendTextReply("Hello everyone");
});
```
